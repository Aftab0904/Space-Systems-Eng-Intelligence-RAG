# Systems Engineering Intelligence RAG (Public-Docs-Edition)

This tool "includes NASA source material" (specifically SP-2016-6105 Rev2) provided under public domain.

This project is a high-performance **Retrieval-Augmented Generation (RAG)** system designed to handle the structural challenges of complex technical documentation, specifically utilizing the public domain **NASA Systems Engineering Handbook**.

## 🚀 Key Features
- **Hierarchical Context:** Preserves parent-child relationships between technical sections (e.g., Section 6.3.2.1).
- **Multi-Hop Reasoning:** Connects logic across different chapters (e.g., connecting System Design to Verification).
- **Acronym Resolution:** Automatically handles NASA-specific shorthand (TRL, KDP, SRR).
- **Local Embeddings:** Uses `BAAI/bge-small-en-v1.5` for fast, local processing.
- **Groq Acceleration:** Powered by `Llama-3.3-70b-versatile` for ultra-fast technical synthesis.

## 🛠️ Stack
- **Framework:** LlamaIndex
- **LLM:** Groq (Llama 3.3 70B)
- **Embeddings:** Local CPU (HuggingFace)
- **Vector DB:** ChromaDB
- **UI:** Vanilla CSS/JS Dashboard

## 🚦 Getting Started
1. **Configure Environment:** Add your `GROQ_API_KEY` to `backend/.env`.
2. **Setup:**
   ```powershell
   cd backend
   .\venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. **Launch:**
   ```powershell
   python main.py
   ```
4. **Deploy:** Open `http://localhost:8000` and click "Deploy Dataset" to index the public handbook.
